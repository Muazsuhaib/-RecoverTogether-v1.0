## Overview
RecoverTogether is an iOS accountability app that helps people 
overcome social media addiction through real-time community 
monitoring and group accountability.

## Core Feature — Real-Time Notifications
- 15-Min Overdue Alert — group notified if you miss a task
- Left During Task — group alerted if you leave the app
- Task Completed — group celebrates when you finish
- Quick Reply — auto-generated message with one tap Send button

## Features
- Community dashboard with color coded status
- Discord-style group chat
- Friends system
- Weekly calendar with task tracking
- Leaderboard with real data
- Focus sessions
- Mood tracker
- Streak badges
- Dark/light mode

## Installation
1. Clone this repository
2. Open in Xcode 15 or 16 or 26
3. Add GoogleService-Info.plist to project root
4. Add Firebase via File > Add Package Dependencies
5. Select FirebaseAuth, FirebaseFirestore, FirebaseMessaging
6. Press Cmd + R to build and run

## Tech Stack
- Swift / SwiftUI (iOS 16+)
- Firebase Auth
- Firebase Firestore
- Firebase Messaging
- Xcode 15/16
